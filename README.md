# Cybersecurity Awareness Chatbot

## Project Description
This is a C# console-based chatbot designed to educate users about cybersecurity.  
The chatbot provides guidance on password safety, phishing, and safe browsing.

## Features
- Voice greeting using a WAV file
- ASCII art logo displayed at startup
- Personalised user interaction (asks for user’s name)
- Input validation to prevent empty inputs
- Chatbot responds to keywords like "password", "phishing", "safe browsing"
- Typing effect for realistic conversation

## How to Run
1. Open the project in Visual Studio.
2. Make sure `greeting.wav` is added to the project and set to **Copy if newer** in Properties.
3. Press **F5** or click **Start** to run the program.
4. Enter your name when prompted and start chatting with the bot.

## Example Questions
- "How are you?"
- "What is your purpose?"
- "Tell me about passwords"
- "What is phishing?"
- "Safe browsing tips"

## Project Structure
- `Program.cs` → Main chatbot logic
